# Portfolio

This is the source code for my personal portfolio website.

**Live Site:** [https://andylin9872.github.io/my-portfolio/](https://andylin9872.github.io/my-portfolio/)

## Tech Stack

- **[Astro](https://astro.build)**
- **[Tailwind CSS](https://tailwindcss.com/)**
- **[DaisyUI](https://daisyui.com/)**

## Local Development

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/andylin9872/my-portfolio.git
    cd my-portfolio
    ```

2.  **Install dependencies**:
    ```bash
    pnpm install
    ```

3.  **Run development server**:
    ```bash
    pnpm run dev
    ```
