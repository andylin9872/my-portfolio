## Overview

Developed a browser extension integrating with Google Street View to visualize astronomical data.

## Key Features

- **Browser Extension**: Integrates seamlessly with Google Street View.
- **Rendering Algorithms**: Implemented Field of View (FOV) overlays and star trail rendering algorithms on panoramic images.
- **User Interface**: Engineered a user-friendly interface for real-time interaction with spatial data overlays.

## Technologies

- JavaScript
- Chrome Extension API
