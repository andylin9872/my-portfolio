## Overview

Built a custom Electrical Impedance Tomography (EIT) hardware system for dynamic muscle sensing.

## Key Features

- **Hardware Design**: Custom EIT hardware system.
- **Signal Processing**: Developed signal reconstruction pipelines to convert raw impedance data into real-time activity maps.
- **Real-time Visualization**: Achieved real-time visualization of muscle contraction in controlled motion experiments.
